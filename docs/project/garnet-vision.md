# Garnet 비전 및 방향

## 핵심 비전

Garnet은 단순한 마케팅 자동화 도구가 아닌, 지속적으로 학습하고 개선하는 **개인 AGI 에이전트 시스템**이다.

### 원칙
1. **개인 자산** — 특정 회사에 종속되지 않는 독립 시스템
2. **장기 진화** — 지속적으로 발전하는 구조
3. **구조 중심** — 에이전트 구조 단위로 사고
4. **승인 기반 자율** — 핵심 변경은 인간 승인 기반
5. **병렬 처리** — 24시간 자율 탐색/분석/제안

## 현재 상태 (v0.5.0+)

### 완료된 핵심 축
- 마케팅 OS 기반 (19+ 페이지, 프리미엄 UI)
- Cron 스케줄러 (12개 잡, 24시간 자동 실행)
- 마케팅 인텔리전스 (5개 플랫폼 수집 + AI 분석)
- GA4 성과 분석 (실데이터, 11개 섹션 대시보드)
- Instagram 연동 (OAuth + 인사이트)
- 영상 생성 (fal.ai LTX + 스크립트)
- AI 코파일럿 (Cmd+. 마케팅 전문 채팅)
- Agent Shell (JARVIS형 커맨드 인터페이스)
- **Flow Builder** (에이전트 파이프라인 비주얼 에디터)
- **Gemma 4 통합** (무료 최상위 LLM 프로바이더)
- **Shell <> Flow 자동 생성/실행** (자연어 -> 에이전트 팀 자동 설계)

### Self-Improvement 로드맵
| 단계 | 현재 매핑 | 상태 |
|------|----------|------|
| Scout (탐색) | 마케팅 인텔 수집기, Tech Radar | 부분 완료 |
| Analyst (분석) | AI 분석, GA4 해석 | 부분 완료 |
| Builder (초안) | Flow Builder, 영상 스크립트 | 완료 |
| Governor (승인) | 승인 인박스 | 부분 완료 |

## 제품 방향
- 사용자에게 업무 언어를 보여준다 (기술 용어 X)
- 대시보드/카드/우선순위/승인 액션 우선
- 워크플로우 중심 설계
- 핵심 흐름: 브리핑 -> 캠페인 룸 -> 세미나/플로우 실행 -> 보고서/플레이북

> **향후 확장 메모:** Self-Improvement 4단계 루프는 `workflows/self-improvement-loop.md`로 분리 가능
