# Garnet 로드맵

- Cron 스케줄러 (12개 잡)
- 마케팅 인텔리전스 (5개 플랫폼)
- GA4 성과 분석

1. Supabase SMTP 차단 해제 (Resend/SendGrid)
2. Instagram Login 전환
3. Flow 실행 결과 -> 세미나 연동
4. Self-Improvement 루프 구체화 (Scout -> Analyst -> Builder -> Governor)

### Phase 2 — 단기
5. MCP 레지스트리 확장 (Notion, Slack, GDrive, Brave Search)
6. Supabase 데이터 이전 Phase 1-2
7. Telegram 게이트웨이
8. Research Memory (지식 축적)

### Phase 3 — 중기

10. Analytics Forecast (예측 분석)
