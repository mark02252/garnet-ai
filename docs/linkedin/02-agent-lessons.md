# Post 2: 에이전트가 스스로 배운 것

## 본문

AI 에이전트가 스스로 배운 것 중 인상적이었던 3가지.

Garnet이라는 마케팅 에이전트를 만들어서 운영하고 있는데,
이 에이전트는 매시간 자기가 내린 판단을 되돌아보고
교훈을 스스로 축적합니다.

사람이 가르친 게 아니라,
에이전트가 데이터를 보고 스스로 정리한 겁니다.

━━━━━━━━

1. "도달률은 높은데 참여율이 낮으면, 노출을 늘리는 게 아니라 인터랙티브 요소를 넣어야 한다"

→ 단순 노출 확대로는 반응이 안 온다는 걸
에이전트가 반복 분석 끝에 스스로 결론 내렸습니다.

2. "과거에 거절된 전략도 시즌이 맞으면 다시 꺼내야 한다"

→ 봄철 B2B 수요가 올라오는 시점에
이전에 보류됐던 기업 대관 전략을 다시 제안했습니다.
타이밍이 전략의 절반이라는 걸 데이터로 배운 겁니다.

3. "트래픽이 정체될 때는 신규 유입보다 전환율 최적화가 효율적이다"

→ 방문자 수는 충분한데 전환이 안 되는 시점을 감지하고,
유입 전략 대신 CRO(전환율 최적화)를 우선 제안했습니다.

━━━━━━━━

이 교훈들이 3회 이상 반복 검증되면
"원칙"으로 자동 승격되어 이후 모든 판단에 반영됩니다.

에이전트가 경험에서 배우고,
배운 걸 다음 판단에 쓰는 구조입니다.

마케터가 하루에 110번 판단을 내릴 수는 없지만,
에이전트는 합니다. 그리고 매번 더 나아집니다.
